# 大物实验报告 Skill

## 简介

这是一个帮助华南理工大学学生自动生成大学物理实验报告的 AI Skill。它能够根据 PPT 课件和原始数据图片，自动生成符合学校格式要求的 LaTeX 实验报告，并编译为带水印的 PDF 文件。

## 功能特点

- 自动读取 PPT 课件内容提取实验信息
- 根据原始数据图片自动填入实验数据
- 生成符合华南理工大学格式要求的 LaTeX 报告
- 自动编译 PDF 并添加学校水印
- 按实验名称自动命名文件，防止覆盖

## 前置安装要求

### 1. MiKTeX 安装

1. 访问 [MiKTeX 官网](https://miktex.org/download) 下载并安装
2. 安装时选择"Complete"完整安装模式
3. 安装完成后，将 MiKTeX 的 bin 目录添加到系统 PATH 环境变量
   - 默认路径：`D:\Miktex\miktex\bin\x64` 或 `C:\Program Files\MiKTeX\miktex\bin\x64`

### 2. VS Code 扩展安装

在 VS Code 中安装以下扩展：
- **LaTeX Workshop**：用于 LaTeX 代码编辑和编译
- **LaTeX Language Support**：提供 LaTeX 语法支持

### 3. Python 依赖

确保已安装 Python 3.x，并安装以下库：
```bash
pip install python-pptx pywin32
```

## 使用方法

### 基本使用

1. **准备材料**：
   - 将实验 PPT 课件放在指定目录：`C:\Users\Cgy123456\Desktop\学长的焚决\大一下\大物实验\`
   - 准备好原始数据图片

2. **触发 Skill**：
   在对话中输入以下任一关键词：
   - "帮我完成大物实验报告"
   - "完成大物实验报告"
   - 上传 PPT 课件或原始数据图片并提及要写实验报告

3. **选择实验**：
   Skill 会自动列出可用的 PPT 课件，选择需要完成的实验

4. **生成报告**：
   - Skill 会自动读取 PPT 内容和原始数据
   - 生成 LaTeX 代码文件（.tex）
   - 自动编译为 PDF 文件

5. **查看结果**：
   生成的 PDF 文件保存在：`C:\Users\Cgy123456\Desktop\学长的焚决\大一下\大物实验\大物实验报告\`

### 文件命名规则

- LaTeX 源代码：`[实验名称].tex`
- PDF 报告：`[实验名称].pdf`
- 例如：PPT 名为"金属逸出功.ppt"，则生成"金属逸出功.tex"和"金属逸出功.pdf"

## 报告格式

### 封面
- 华南理工大学 Logo
- 实验报告标题
- 实验名称
- 个人信息表格（课程名称、姓名、学号、专业班级、开课学期、实验日期）
- 底部：物理与光电学院公共教学实验中心 + 年月

### 正文
1. 实验名称
2. 引言
3. 一、实验目的
4. 二、实验仪器
5. 三、实验原理
6. 四、实验内容与步骤
7. 五、数据处理
8. 六、实验结果与分析
9. 七、思考题（仅当 PPT 中包含时添加）

### 封底
- 原始数据记录表格
- 实验者签名和日期区域

## 注意事项

1. **数据准确性**：Skill 严格遵循原始数据，不会自行编造或修改数据
2. **留空原则**：无法从 PPT 或图片中提取的内容会标注【待补充】
3. **格式规范**：正文使用宋体小四号，1.5 倍行距
4. **公式环境**：所有公式使用标准 LaTeX 数学环境
5. **文件覆盖**：按实验名称命名文件，不会覆盖之前的报告

## 故障排除

### 常见问题

1. **PPT 无法读取**：
   - 确保 PPT 文件放在正确目录
   - 旧版.ppt 文件会自动转换为.pptx 格式

2. **LaTeX 编译失败**：
   - 检查 MiKTeX 是否正确安装
   - 确认 xelatex 命令在 PATH 中可用

3. **PDF 未生成**：
   - 检查输出目录是否存在
   - 确认有足够的磁盘空间
